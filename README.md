# Tooth-Kingdom-DAAI
## Overview

Tooth Kingdom is a gamified digital health application designed to encourage healthy brushing habits among children aged 4–8 years.

This project demonstrates how Artificial Intelligence and Data Analytics can be integrated into digital health by predicting children's risk of brushing nonadherence using a Random Forest classifier.


## Machine Learning Task

Predict:

- Low Risk
- Medium Risk
- High Risk

Using behavioral and gamification data collected within Tooth Kingdom.


## Algorithm

Random Forest Classifier


## Dataset

The dataset contains 1000 synthetic behavioral records generated for academic proof-of-concept purposes.

No real patient or child data were used.
